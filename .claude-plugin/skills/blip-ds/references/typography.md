| --- | --- | --- |
| `fs-40` | `2.5rem` | `40` |
| `fs-32` | `2rem` | `32` |
| `fs-24` | `1.5rem` | `24` |

## Usage Notes
- Prefer `bds-typo` to keep typography consistent.
- Use semantic tags via `bds-typo` `tag` prop for accessibility.
